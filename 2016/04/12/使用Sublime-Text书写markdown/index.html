安装sublime text

sublime text是一款简单，强大的文本编辑器
下载地址

安装sublime的插件管理工具：Package Control

安装方法(访问需翻墙)

作者使用的是手动安装方法

  • 将Package Control.sublime-package下载到本地。
  • 然后将文件拷贝到sublime下的Installed Packages中,如果你的版本是sublime text 2,则下面的目录应该把3改成2
  • cp Package\ Control.sublime-package ~/Library/Application\ Support/Sublime\ Text\ 3/Installed\ Packages
  • 如果出现下图的情况，则安装成功

    使用Package Control安装插件markdown preview

  • 打开上图中的Package Control (command+Shift+P)
  • 输入Install Package,敲击回车，稍等片刻
  • 输入markdown preview，回车。安装成功。
  • 用sublime编写.md后缀的markdown文件，使用command+B会在当前目录下生成.html文件

在终端下，如何使用sublime text打开文本？

vi ~/.bash_profile
在文件中加入这句：alias subl='/Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl'
source ~/.bash_profile
subl testfile.txt

markdown介绍

可以用任何编辑器来书写使用Markdown格式的文件，它是纯文本文件但是却能使文本内容拥有美观的格式，文件名使用.md后缀。
.md文件可以被翻译成html、pdf文件。
格式语法：www.markdown.cn
